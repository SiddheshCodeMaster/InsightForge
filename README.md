# InsightForge

InsightForge is an autonomous business intelligence system that analyzes structured datasets and generates actionable insights using a multi-agent AI architecture.

## Features

- Automated dataset profiling
- Revenue trend detection
- Hypothesis generation
- Statistical validation
- Executive insight reports

## Architecture

ProfilerAgent → TrendAgent → HypothesisAgent → StatsAgent → InsightAgent → ReportAgent

## Usage

CLI

python -m cli.run_analysis

API (coming soon)

POST /analyze
